# free_test_apis
收集网上一些免费使用的测试api接口
前端需要的免费在线api接口: https://juejin.cn/post/7041461420818432030
### 帖子接口：

获取帖子列表：http://jsonplaceholder.typicode.com/posts <br>
根据帖子ID获取详情：http://jsonplaceholder.typicode.com/posts/1 <br>
获取某个用户所有的帖子：http://jsonplaceholder.typicode.com/posts?userI…  <br>
获取帖子所有的评论：http://jsonplaceholder.typicode.com/posts/1/com…  <br>
